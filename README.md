# project-structure 项目结构
本项目构建一个认为合理的项目结构,不断学习,持续更新！！！



## 项目结构

- controller 接口实现
- service 业务实现
- dao 数据库实现
- entity 实体类
    - po 数据库交互实体类
    - vo 前后端交互实体类
    - dto 远程服务接口调用实体类
    - query 查询接口调用实体类
    - form 调用保存/修改 接口实体类
- config 配置文件
- utils 工具类

## utils 工具类
